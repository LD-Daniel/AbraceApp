# AbraceApp

A ideia deste aplicativo surgiu a partir da observação de uma necessidade real dentro da ONG Abrace: a automatização de processos logísticos que antes eram feitos manualmente.

Anteriormente, os caminhoneiros precisavam preencher informações sobre suas viagens em formulários de papel. Esse método trazia diversos problemas, como perda de tempo, risco de extravio dos papéis e dificuldades para os supervisores entenderem a escrita manual e posteriormente digitarem os dados no sistema.

Para resolver esses desafios, o app foi desenvolvido com foco na praticidade e acessibilidade. Ele conta com uma tela inicial de login, que oferece as opções "Esqueci minha senha" e "Não estou cadastrado", garantindo suporte a diferentes situações de acesso.

Após o login, o usuário é direcionado para uma área específica conforme seu perfil:

* Caminhoneiros: têm acesso a um formulário otimizado para preenchimento rápido, claro e simples, pensado para ser o mais acessível possível no dia a dia da estrada.

* Supervisores: são redirecionados para um painel com todos os formulários preenchidos, contendo informações essenciais como data, hora, nome do colaborador, entre outros dados relevantes para a gestão.

Se o login ou a senha estiverem incorretos, uma mensagem de erro será exibida em vermelho logo acima do campo de entrada, informando o problema de forma clara.
